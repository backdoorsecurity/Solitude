Solitude is a lightweight, Debian operating system configuration designed for Linux power users seeking the security model of Qubesos.

Built with Sway as the window manager and KVM/QEMU for virtualization, it replaces Xen with a simpler, much lighter stack.

The system emphasizes a minimal browser and net VM, allowing users to isolate untrusted workloads efficiently. Solitude aims to reduce redundancy and complexity while preserving strong security boundaries, making advanced isolation efficient and seamless.
